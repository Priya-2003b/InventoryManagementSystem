# InventoryManagementSystem
Inventory Management System is a web-based application built using PHP and MySQL for backend processing, and JavaScript, HTML, and CSS for the frontend. It helps businesses manage products, track stock levels, record sales and purchases, and generate reports efficiently.

A web-based Inventory Management System developed using PHP, MySQL, JavaScript, HTML, and CSS. This system helps businesses track products, manage stock levels, and monitor inventory transactions efficiently.

## Features
- User authentication and role management
- Add, edit, and delete products
- Real-time stock tracking
- Purchase and sales management
- Inventory reports
- Responsive user interface

## Technologies Used
- Backend: PHP
- Database: MySQL
- Frontend: HTML, CSS, JavaScript
- Server: Apache (XAMPP/WAMP)

## Installation
1. Clone the repository or download the project files  
2. Move the project folder to `htdocs` (for XAMPP)  
3. Import the provided SQL file into MySQL database  
4. Update database configuration in `config.php`
